# Rss reader

Using php to read a rss feed as json

## Installation

Use the package manager [composer](https://getcomposer.org/) to install the app.

```bash
composer install
```

## Usage

```php
use method singleChannel($channel) for a single rss feed or

multipleChannels($ch1, $ch2) for multiple rss channels
```

## Run
In terminal

```bash
php index.php
```
